A、B、C、D类:




广播地址：
广播地址应用于网络中所有的主机。
 分为两种:
	(1)受限广播:它不会被路由发送，但会被送到相同物理网络断中的所有主机。(255.255.255.255)
	(2)直接广播:网络广播会被路由，并会发送到专门网络上的每个主机
		ip地址: 主机段全为1 。  如 192.168.10.255 为192.168.10.0/8网络内的直接广播地址 
	
D类IP地址: 
	第一个字节已11100000开始(224),到11110000(240)结束,地址覆盖为 224.0.0.0 ~ 239.255.255.255.
	D类地址不分网络地址和主机地址
