XML语法：
	XML文件一定是以.xml文件结尾
	xml文件需要使用xml解析器
	1.标签
		语法：	<student>
					student
				</student>
		(1)空标签:	<student/> or
					<student></student>	
		(2)xm标签名称区分大小写
		(3)xml标签一定要正确配对。
		(4)xml标签名中不能使用空格。
		(5)xml标签名不能以数字开头
	
	2.属性
		语法：<Student name="name">student</Student>
		(1)属性值必须以引号包含，并且不能单引号、双引号混用。
		(2)一个标签内可以有多个属性，但是不能出现重复的属性名称

	4.文档声明
